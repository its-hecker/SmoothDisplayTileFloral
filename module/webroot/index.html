Notes

Battery saver: When battery saver is active, Android may cap the display to 60Hz regardless of the mode set here. Disable battery saver for Force 90Hz to take full effect.

Coral (Pixel 4 XL) supports 90Hz at any brightness. Flame (Pixel 4) may still drop at very low brightness — hardware limit.

QS tile cycles Auto → 60Hz → 90Hz → Auto on each tap. On first tap after install, approve the root access prompt from ReSukiSU — required once so the tile can read and write the mode config.

Changes apply live — no reboot needed. Mode also persists across reboots.
